## 五、使用指南

### 5.1 论文撰写流程

1. **第一步**: 阅读 `paper_structure.md` 了解论文整体结构
2. **第二步**: 参考 `paper_draft.md` 撰写具体内容
3. **第三步**: 使用 `paper_references.md` 添加引用
4. **第四步**: 根据 `paper_highlights.md` 撰写亮点
5. **第五步**: 使用 `paper_submission_checklist.md` 准备投稿

### 5.2 实验数据填充

待实验完成后，在 `paper_draft.md` 中搜索 `[待填充]` 标记，填入实际实验数据：

```bash
# 搜索待填充内容
grep -n "待填充" paper_draft.md
```
